## Installation

Clone the repository and run the install script:

   ```bash
   git clone https://github.com/catalincd/PacketTracer
   cd PacketTracer
   npm install
   ```

## Run the spell
   ```bash
   npm run start
   ```

## How to use
Load a file, wait for it to parse, then click on what stream, device or packet you want to inspect.
